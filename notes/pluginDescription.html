Tarantool Database Plugin

Support SQL for Tarantool DB.

It's first release of non-official plugin for support Tarantool DB.

This plugin based on default API supported SQL92 provided by IDEA.

That means plugin doesn't support specific properties of Tarantool SQL/DB at this moment.

If you need more information about real syntax and features of Tarantool SQL you can read page on official Tarantool WEB
